# Logistics Route Optimizer
A lightweight, dependency-free route optimization tool built for logistics planning.

## Overview
This script implements Dijkstra’s algorithm to calculate the most efficient path between distribution hubs. Designed as a standalone utility for high-speed logistics routing.

## Features
- **Zero Dependencies:** Uses native Python libraries only.
- **Efficient Pathfinding:** Implements a priority-queue-based search for optimal performance.
- **Modular Design:** Easily extensible to include traffic variables or fuel costs.

## Usage
Simply run the script to see a demonstration of the shortest path calculation:
`python logistics_optimizer.py`
